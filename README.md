# 🚚 Move It – Modern Moving Service Website

Welcome to **Move It**, a responsive and modern moving service website built with **Bootstrap 5** and **custom CSS animations**.  
It’s designed to showcase a moving company’s professionalism, services, and reliability in a sleek, dark-themed layout.

---

## 🌐 Live Demo
👉 [View Website](https://pranjal-glitch.github.io/move-it/)

---

## ✨ Features

- 🧭 Responsive navigation bar with branding  
- 🎯 Hero section with tagline and CTA buttons  
- 🚛 “Why Move With Us” feature section with icons  
- 🖼️ Smooth fade-in animations  
- 📸 Bootstrap image carousel with captions  
- 📬 Stylish footer with newsletter signup  
- ⚡ Fully responsive and mobile-friendly  

## 🧱 Built With

- [Bootstrap 5.3](https://getbootstrap.com/)  
- Custom CSS (for fade-in effects and colors)  
- HTML5  
- JavaScript (for scroll-based animations)
